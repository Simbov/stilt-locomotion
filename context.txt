WHAT THIS PROJECT IS
--------------------
Training a Unitree G1 humanoid robot to walk on physical stilts (220x80x400 mm aluminum
extrusion with a flat base). Using mjlab (Isaac Lab API + MuJoCo Warp) and PPO via rsl-rl-lib.
Training runs on QUT Aqua HPC (H100 GPU). Local dev on MacBook M4 (CPU only).

STILT DESIGN IN MJCF
---------------------
- Stilts attach as rigid bodies (no joint) to left/right ankle_roll_link
- Stilt mesh: refpos="70 40 435" (mm, pre-scale) aligns attachment face with ankle origin
- 8 collision capsules per stilt (4 left block + 4 right block), all at z=-0.425, radius=0.01
  → bottom of capsules at z=-0.435 m from ankle = stilt tip
- foot_capsule default class: friction="1.0 0.005 0.0001" condim="3" (explicit, not relying on CollisionCfg)
- Original G1 foot capsules commented out
- Stilt tip sites: left_stilt_tip / right_stilt_tip at pos="0.04 0 -0.435"
- Pelvis spawn in MJCF: z=1.228 (standing straight). Training uses z=1.16 via keyframe (bent knees).
- Kinematics verified: stilt tips are 3 mm above floor at spawn (correct, drops and lands)

---
ENV CONFIG KEY SETTINGS (envs/stilt_g1/env_cfgs.py)
----------------------------------------------------
- site_names = ("left_stilt_tip", "right_stilt_tip")
- geom_names = 16 stilt capsule names (left/right_stilt_l1-l4/r1-r4_collision)
- foot_clearance target_height = 0.10 m (same as stock G1 — don't increase yet)
- foot_swing_height target_height = 0.10 m
- air_time weight = 0.0 (disabled until robot walks)
- torso_too_low threshold = 0.65 m (fires if pelvis drops below this)
- foot_friction randomisation targets stilt capsule geoms

CRITICAL LESSON: threshold was 0.85 m, which caused every episode to end in 13 steps
(free-fall time from 1.16 m spawn to 0.85 m). This prevented any learning for 1500 iters.
Fixed to 0.65 m.

---
COLLISION SETUP (envs/stilt_g1/stilt_robot.py)
-----------------------------------------------
CollisionCfg targets ONLY stilt geoms:
  geom_names_expr = (r"^(left|right)_stilt_[lr][1-4]_collision$",)
  condim=3, priority=1, friction=(1.0, 0.005, 0.0001)

disable_other_geoms=True (CollisionCfg default) disables all non-stilt geoms.
This is fine — only stilts touch the ground.

Spawn keyframe: pos=(0, 0, 1.16), same joint angles as stock G1 KNEES_BENT_KEYFRAME.
Calculation: stock G1 bent-knee spawn = 0.76 m. Stilts add 0.40 m extra ground clearance.
0.76 + 0.40 = 1.16 m.

KEY MJLAB INTERNALS
--------------------
Curriculum (in velocity_env_cfg.py):
  iter 0:      lin_vel_x ±1.0 m/s,  ang_vel_z ±0.5
  iter 5000:   lin_vel_x -1.5→+2.0, ang_vel_z ±0.7
  iter 10000:  lin_vel_x -2.0→+3.0
  (step * 24 because 24 mini-batches per gradient iteration)

CollisionCfg behaviour:
  - geom_names_expr uses re.match() (NOT full string match, but ^ and $ in pattern work)
  - disable_other_geoms=True (default) → sets contype=0,conaffinity=0 on non-matched geoms
  - This is fine for us — only stilt capsules need floor contact
  - G1_ARTICULATION does NOT contain collision config (only actuators)
  - FULL_COLLISION is in get_g1_robot_cfg(), not in G1_ARTICULATION

MJCF refpos:
  - refpos is in pre-scale units (mm when scale="0.001 0.001 0.001")
  - Formula: body_coord (m) = (stl_coord_mm - refpos_mm) × 0.001
